Forms


I'm just going to code all of the types of form I can,
I'll make the lines visible too:

  1. Text input:
    Username:
    Looks like this:
    <form action="http://www.example.com/login.php">Username:<input type="text" name="username" size="15" maxlength="30" /> </form>

  2. Password input:


    Looks like this:
    <input type="password" name="password" size="15" maxlength="30" />

  3. Textarea input:


    Looks like this:
    <textarea name="comments" cols="20" row="4">Answers here...</textarea>

  4. Radio input:
    Rock Pop Jazz

    Looks like this:
    <input type="radio" name="genre" value="rock" check="checked" /> Rock
    <input type="radio" name="genre" value="pop" check="checked" /> Pop
    <input type="radio" name="genre" value="jazz" check="checked" /> Jazz

  5. checkbox input:
    iTunes Last.fm Spotify

    Looks like this:
    <input type="checkbox" name="service" value="itunes" checked="checked" /> iTunes
    <input type="checkbox" name="service" value="lastfm" checked="checked" /> Last.fm
    <input type="checkbox" name="service" value="spotify" checked="checked" /> Spotify

  6. Select (Drop down menu) input:

    What device do you surf the internet on most?


    Looks like this:
    <select name="devices">
    <option value="smart phone">Smartphone</option>
    <option value="computer">Computer</option>
    <option value="telepathic squid-monster">Telepathic Squid-Monster</option></select>

  7. Multiple Select (Drop down menu) input:

    What instruments do you play?
    You can choose multiple options (Hold cmd)


    Looks like this:
    <Select name="instruments" size="5" multiple="multiple">
    <option value="guitar" selected="selected">Guitar</option>
    <option value="bass" selected="selected">Bass</option>
    <option value="drums" selected="selected">Drums</option>
    <option value="trumpet" selected="selected">Trumpet</option>
    <option value="sitar" selected="selected">Sitar</option>

  8. File input:

    Upload your song in MP3 format:



    Looks like this:
    <input type="file" name="user-song" /><br />
    <input type="submit" value="Upload" />

  9. Submit button input:




    Looks like this:
    <input type="text" name="email" /><br />
    <input type="submit" name="subscribe value="Upload" />
