Adding HTML5 Audio:



Looks like this:

<audio src="../aud/test-audio.ogg" controls="controls">

You can edit all sorts by adding attributes, we'll look into that in-depth later.
You can also provide multiple sources for audio since different browsers support differnet formats.

Looks like this:
<audio>
<src="aud/test-audio.ogg"> <src="aud/test-audio.mp3">
</audio>

Still pretty damn easy!
